## Top 5 Fixes Prioritaires

### 🔴 1. **Perk picker UI + Hero progression loop** (30 min)
- **Why** : Hero.Level increments but no UI to select perks → player confusion, XP dead mechanic.
- **Impl** : Create PerkPickerPanel.cs (UI Toolkit), wire Hero.OnLevelUp event, populate 3-choice dropdown from PerkSystem.
- **Files** : `Assets/Scripts/UI/PerkPickerPanel.cs` (new), `Hero.cs` (event hook), `LevelRunner.cs` (subscribe).

### 🔴 2. **Castle HP parametric balance** (20 min)
- **Why** : Hardcoded 120 HP for POC, but D1-04 spec = `100 + 50×√(world) × diffMul` not parametrized.
- **Impl** : Move `CASTLE_HP` constants to BalanceConfig.cs SO asset, apply formula in Castle.Init().
- **Files** : `Assets/Scripts/Systems/BalanceConfig.cs` (extend), `Castle.cs` (init logic).

### 🟠 3. **Boss visual aura + enrage feedback** (25 min)
- **Why** : Enemy enrage flag set but no particle + audio feedback → player unaware.
- **Impl** : In BossSystem.TriggerEnrage(), spawn VFX.SpawnBossAura(enemyPos) + audio, add outline color pulse.
- **Files** : `Assets/Scripts/Systems/BossSystem.cs`, `VfxPool.cs` (new particle), `Enemy.cs` (visual hooks).

### 🟠 4. **Interest / bank savings mechanic** (35 min)
- **Why** : D1-01 economy rework = 5% interest on excess coins, but Economy.cs has TODO comment.
- **Impl** : Add `bankCoins` field, apply interest calc each wave-end, show "+X% interest" toast.
- **Files** : `Economy.cs` (calc), `WaveManager.cs` (wave-end hook), `ToastManager.cs` (UI feedback).

### 🟡 5. **Magnet slow aura effect** (15 min)
- **Why** : Magnet tower has \_pullActive flag but enemies not slowed by proximity (0.7× speed).
- **Impl** : In Synergies.Resolve(), apply SlowEffect to enemies within magnet range when \_pullActive.
- **Files** : `Synergies.cs` (apply slow to enemies), `SlowEffectManager.cs` (hook).
