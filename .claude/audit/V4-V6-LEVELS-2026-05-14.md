## 0. TL;DR

**V4 Crowd Defense (`src-v3/`)** = 80 levels campagne (10 mondes × 8) + boss_rush + endless + 10 debug levels + 3 testlevels (showcase, mazetest, streamtest). Format JS object littéraux. Grammar grid 0/1/P/C/~/^/W/L/T/R/D/B/space, multi-portail + multi-château possible.

**V6 (Unity 6, `crowd-defense/`)** = **90 levels campagne** (10 mondes × 9 — V6 a ajouté 1 mega-level final par monde, W*-9) + Endless procédural runtime + Daily seed. Format ScriptableObject (`.asset` YAML). Grammar identique au V4 + 1 ajout : `*` = treasure tile.

**V4 Park Defense Phaser (`src-v2/`)** = jeu PvZ-style ligne-à-ligne, complètement différent (66 levels = 60 campagne + 5 carnival conveyor + bossarena + endless + daily). **HORS SCOPE refonte** : ce code est sur tag `v1.0-kaplay-final`/`v3.0-park-defense-stable`, archivé.
