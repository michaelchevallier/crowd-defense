# W5 — Foire (Plateau-festif)

**Theme** : `foire`
**Levels** : W5-1 (Entrée du Parc) → W5-7 (Tour-volcan) → W5-8 (Boss : Apothéose Carnaval)
**Castle HP range** : 195-260
**Mob roster** : Basic, Runner, Brute, Flyer, **Shielded** (intro)
**Wave count** : 4-9 (le world le plus chargé — apothéose mid-jeu)

## Lore-light

Une foire éclatante de couleurs, mais quelque chose cloche : les manèges tournent vides, la grande roue grince. Les forains sont devenus des choses, leurs masques figés en sourires terrifiants. Le carnaval ne s'arrête jamais.

## Narrative hook

W5 = **point culminant du milieu de jeu**. C'est la "dernière respiration" avant la rupture W6. Le joueur doit avoir maîtrisé toutes les bases (tours L1/L2, économie, magnet, synergies basiques). Tous les mob types W1-W4 reviennent en mix dense.

## Gameplay focus

- **W5-1 à W5-3** : densification waves (3-4 mob types en parallèle). Châteaux fixés à ratio 0.92-0.96 spec (cf gap-report).
- **W5-4 à W5-5** : Shielded mob — résistance frontale aux DPS basique. Need acid/utility pour break.
- **W5-6** : pic de pression — **1200 mobs total** sur 8 waves. Stress test économique.
- **W5-7** : 8 waves, mix complet. Prépare le joueur au volume W6+.
- **W5-8 Boss WarlordBoss "Apothéose Carnaval"** : 9 waves, boss + escortes. Pic culminant pré-rupture.

## Décor

- Sol : pavé multicolore + bordures parc (`#8a4a8a` palette saturée)
- Eléments : manèges en arrière-plan (tour-grande-roue, carrousel cassé), bouquets de ballons
- Castle : tente royale du parc / chapiteau renforcé
- Ennemis : silhouettes carnavalesques (clowns, mascottes corrompues), forain warlord boss
