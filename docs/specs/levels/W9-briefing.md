# W9 — Royaume (Magie-classique)

**Theme** : `medieval`
**Levels** : W9-1 (Frontière) → W9-7 (Donjon central) → W9-8 (Boss : Sorcier-Roi)
**Castle HP range** : 250-400
**Mob roster** : SkeletonMinion, Brute, Flyer, **WizardKing** (boss)

## Lore-light

Le château est entré dans un royaume fantastique où la magie est dense, ancienne. Des sorciers conjurent des squelettes infinis. La frontière franchie, plus de retour possible. Au cœur du royaume, un trône — et son occupant attend.

## Narrative hook

W9 = retour stylistique medieval-fantasy classique après l'éclectisme de W6-W8. Hommage TD pur : squelettes, sorciers, donjons. Le joueur reconnaît les codes.

## Gameplay focus

- **W9-1 à W9-3** : SkeletonMinion spam (mob lent mais nombreux). Domaine AOE tour (Mage, Cannon).
- **W9-4 à W9-5** : Brute médiéval — HP très élevé, lent. Need single-target high-dmg (Crossbow L3).
- **W9-6 à W9-7** : multi-portails 3-4P, vagues massives. Tests les builds late-game complets.
- **W9-8 Boss WizardKing "Sorcier-Roi"** : mob unique + summons squelettes en boucle + sorts AOE. Test multi-pression.

## Décor

- Sol : pierre médiévale + bannières royales (`#5a4a3a` + `#aa3a3a`)
- Eléments : tours guet, ponts levis, statues royales, vitrines magiques
- Castle : donjon royal renforcé fantasy (style très classique TD)
- Ennemis : squelettes en armure, sorciers à capuche, WizardKing trône volant
