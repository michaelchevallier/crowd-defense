# W1 — Plaine (Onboarding)

**Theme** : `plaine`
**Levels** : W1-1 (tutorial) → W1-7 (skill-check) → W1-8 (Boss : Brigand)
**Castle HP range** : 145-200
**Mob roster** : Basic (Grunt), Runner, Brute

## Lore-light

Une plaine herbeuse aux portes du royaume. Le château isolé garde la dernière ligne d'approvisionnement. Des brigands rôdent. Pas encore de magie, pas encore de fantastique — juste des bandits affamés et leurs grosses brutes.

## Narrative hook

Le joueur est le **maître d'œuvre des défenses** d'un château oublié. Pas de personnage incarné, pas de cinématique : le HUD est l'interface, les tours sont les héros. La plaine sert d'apprentissage avant que les vraies horreurs n'arrivent (W6+).

## Gameplay focus

- **W1-1 à W1-3** : poser une tour, lancer une vague, observer l'économie. **Skill = 0**, juste comprendre les boutons.
- **W1-4 à W1-6** : introduction Runner (mob rapide qui contourne) — le joueur découvre que toutes les tours ne valent pas pour tous les enemies. Crossbow vs Archer trade-off.
- **W1-7** : skill-check — boucler le jeu sans recevoir d'hit ? Petit défi early-bird.
- **W1-8 Boss Brigand** : 1 mob HP très élevé, vitesse moyenne. Apprends la pression mono-cible.

## Décor

- Sol : herbe verte mid-saturated (`#5a8a3a` env.)
- Eléments : arbres bas isolés (`T`), pas de stream, pas de bridge
- Castle : modèle medieval simple (donjon carré + créneaux)
- Ennemis : silhouettes humanoïdes brigand (capuchon brun)
