# W3 — Désert (Chaleur-pression)

**Theme** : `desert`
**Levels** : W3-1 (Dunes) → W3-7 (Mirage) → W3-8 (Boss : Épreuve Finale)
**Castle HP range** : 150-200
**Mob roster** : Basic, Runner, Brute, **DesertRunner**, **Assassin**, SkeletonMinion (intro early)

## Lore-light

Les dunes ondulantes cachent des choses anciennes. Le sable est vivant, il glisse sous les pas — les coureurs du désert exploitent ce terrain au mieux. Et certains pas ne laissent aucune trace : les assassins arrivent avec le coucher du soleil.

## Narrative hook

Le désert **introduit l'assassin** (Assassin mob — invisible jusqu'à proximité ou tank). Le joueur doit gérer une menace asymétrique : ses tours ne peuvent pas viser ce qu'elles ne voient pas.

## Gameplay focus

- **W3-1 à W3-3** : DesertRunner spam (vitesse augmentée vs Runner standard).
- **W3-4 à W3-5** : Assassin apparaît — premier mob à statut spécial. Need detector (Mage avec aura ?) ou tank pour révéler.
- **W3-6 à W3-7** : SkeletonMinion preview (W3-8 boss summon swarms de squelettes).
- **W3-8 Boss "Épreuve Finale"** : combinaison Skeleton spawn + Assassin spawn + Brute waves — multi-pression.

## Décor

- Sol : sable ocre + rochers (`#c9a45a` + `#7a5e2c`)
- Eléments : oasis isolées (`T` ou tile spéciale), pas de stream large (juste un wadi `W`)
- Castle : structure pierre désert / fort romain
- Ennemis : silhouettes draped, capuchons clairs, squelettes désertiques
