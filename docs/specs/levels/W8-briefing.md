# W8 — Abysse (Pression-aquatique)

**Theme** : `submarin`
**Levels** : W8-1 (Récif) → W8-7 (Fosse) → W8-8 (Boss : Kraken)
**Castle HP range** : 230-400
**Mob roster** : Basic adapté, Brute, **SubmarinRunner**, **KrakenBoss**

## Lore-light

Le château est sous la mer. Les ennemis nagent dans l'eau noire des abysses. Les rivières `W` deviennent omniprésentes, transformant la map en archipel. Quelque chose remue dans les profondeurs.

## Narrative hook

W8 **utilise W (water/stream) intensivement**. Les bridges deviennent goulots cruciaux. SubmarinRunner exploite les passages aquatiques pour atteindre le château plus vite.

## Gameplay focus

- **W8-1 à W8-3** : intro thème. Maps avec passages mixtes terre-eau.
- **W8-4 à W8-5** : SubmarinRunner — vitesse augmentée sur les portions W. Need fan/frost pour ralentir.
- **W8-6 à W8-7** : multi-passages aquatiques + Brute dense. Pression sur bottleneck.
- **W8-8 Boss KrakenBoss** : mob unique géant + tentacules (multi-targets ?) — combine pression mono + AOE.

## Décor

- Sol : récifs coralliens + sable abyssal (`#1a4a6a` + `#a48a5a`)
- Eléments : épaves de bateaux, coraux fluo, bulles d'air remontant
- Castle : module dome sous-marin, hublots éclairés
- Ennemis : silhouettes monstrueuses aquatiques, kraken titanesque tentaculaire
