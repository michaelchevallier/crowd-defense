# W4 — Volcan (Lave-péril)

**Theme** : `volcan`
**Levels** : W4-1 (Sentier de la Pente) → W4-7 (Caldeira) → W4-8 (Boss : Dragon)
**Castle HP range** : 180-280
**Mob roster** : Basic, Brute, Flyer (généric), Imp, **DragonBoss**

## Lore-light

Le volcan gronde. La lave (`L`) coule en rivières orange qui interdisent le passage — mais où le terrain est dur, des ponts (`^`) ont été construits. Les imps sortent du sous-sol par les fissures. Quelque chose de bien plus grand dort dans la caldeira.

## Narrative hook

W4 **introduit la grammar `L` (lave)** comme barrière vs `W` (eau) du W2-3. Tactiquement : les bridges (`^`) deviennent goulots d'étranglement parfaits pour placer des AOE tours (Mage, Cannon, Mine).

## Gameplay focus

- **W4-1 à W4-3** : intro lave + bridges. Apprends à placer tours sur les bottleneck.
- **W4-4 à W4-5** : Imps (rapide + petits) — saturent les bridges.
- **W4-6 à W4-7** : Flyer générique — survole la lave directement, casse la stratégie pure bottleneck.
- **W4-8 Boss DragonBoss** : mob unique géant + HP énorme + vol — combine toutes les menaces W4. **Cible D1-04 50-70% fail première tentative.**

## Décor

- Sol : roche volcanique noire + lave orange brillante (`#1a1a1a` + `#ff4500`)
- Eléments : rivières de lave (`L`), ponts métalliques (`^`), fissures, geysers fumée
- Castle : forteresse de pierre noire renforcée (medieval gothic dark)
- Ennemis : silhouettes rougeoyantes, imps avec yeux luminescents, dragon final spectaculaire
